## 1. Group Overview

- **이름 의미:** Luminous + Min(민) — 라틴어 + 한국어 글로벌 발음
- **세계관:** 7명의 소년이 각자 다른 도시에서 같은 별을 바라보다 만남. "빛을 잃은 도시에 다시 빛을 가져오는" 컨셉
- **음악:** Synth-Pop + R&B + Future Bass (글로벌 친화)
- **컴백 사이클:** 분기별 (봄 발라드/시티팝 → 여름 댄스/EDM → 가을 R&B → 겨울 캐롤·콘서트)

### 컬러 팔레트
- Primary: `#0F0F1A` (Deep Night)
- Secondary: `#3D5AFE` (Cobalt Blue) / `#FFC857` (Amber Glow)

- 멤버 생일 (월 1–2회) → 단톡방 깜짝 파티

- 컴백 D-30 ~ D-day 카운트다운
- 연말 시상식 함께 보기
- 사용자 생일 → 7명 합창 음성 편지
