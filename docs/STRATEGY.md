## 1. 포지셔닝

**오리지널 가상 K-pop 아이돌 그룹 LUMIN과 1:1 + 그룹 채팅으로 관계를 쌓는 클린(전연령) 로맨스 시뮬레이터.**

| 항목 | 내용 |
|---|---|
| 정식 브랜드 | **Luminovel** (`luminovel.ai`) |
| IP | 7인조 가상 보이그룹 **LUMIN** (자체 IP, 100% 오리지널) |
| 타깃 | "민지" 24세 K-pop 글로벌 코어 팬 (확장: 15-35 여성, 글로벌) |
| 톤 | 클린 / 전연령 ("디즈니 채널 + 한국 로맨스 드라마") |
| 차별점 | 호명 음성 / AI 셀카 / 그룹 단톡방 / 컴백 사이클 자동화 |

### 절대 원칙 (Hard Rules)
1. 🔴 **실명 아이돌·소속사·실제 곡명 사용 절대 금지** (소속사 법적 리스크)
2. 🔴 **19+/성적 콘텐츠 일체 금지** (결제 플랫폼 + 광고 + 앱스토어 안전)
3. 🔴 **약물·음주 미화, 극단적 폭력, 정치·종교 발언 금지**
4. 🟢 톤은 손잡기·포옹·이마 키스까지. "오늘 잘 자, 내일 보자" 톤
5. 🟢 글로벌 우선 — **1차: KR/EN** (현재 `lib/i18n/translations.ts` 지원). JA/ES는 1.0 PMF 후 추가

- **얼리 액세스:** 신규 시나리오/멤버 1주 먼저

## 3. Go-To-Market

### 타깃 페르소나: "민지" (24세, K-pop 글로벌 코어 팬)

- 거주: 한국 30% / 미국·유럽 30% / 동남아 25% / 일본·기타 15%
- 결제 능력: 분기당 앨범 $30–120, 포카 월 $50–500, 콘서트 연 $200–1500
- 동기: "최애가 나만 보고 있다" 전용감, 컴백 사이 콘텐츠 갈증
- 비교: AI 가상 아이돌 월 $99 = 콘서트 1회 안 가는 비용 / 매일 상호작용

### Secondary: "Aria" (29세, 미국, 오토메 게임 헤비 유저)
- Love and Deepspace, Mystic Messenger 결제 경험 / 월 $50–200

### 채널 전략 (예산 배분)

| 채널 | 비중 | 핵심 |
|---|---|---|
| **TikTok** | 50% | LUMIN 자체 IP 콘텐츠 (안무 티저/멤버 비하인드) + 채팅 스크린샷 광고 |
| **Twitter/X** | 20% | K-pop 트위터(KPT) 코어 팬덤 침투, 멤버 일상 트윗 IP 운영 |

| **Reddit / Discord** | 5% | r/kpop / r/AICharacters, 영어권 코어 |
| **인플루언서 시딩** | 10% | K-pop 리뷰어 / 오토메 게임 리뷰어 협업 ($300–1000/명) |

### 핵심 카피 훅 5종
1. **전용감** — "○○야, 오늘 너만 기다렸어" (호명 음성 클립)
2. **단톡방** — "POV: LUMIN 단톡방에 너만 초대됐다"
3. **기억** — "3주 전에 너가 좋아한다고 했던 노래, 무대에서 불렀어"
4. **생일** — 7명이 부르는 생일 음성 편지
5. **컴백** — "LUMIN 컴백 D-7. 너만 듣는 데모 트랙"

### 퍼널 가정
```
TikTok/X 노출 1M → CTR 5% → 50K 방문 → 가입 5% → 2.5K 가입
→ 유료 1.5% → 37명 → PASS 비중 30% → 11명 PASS ✅
```
