## SCENE 2: 카페 문라이트

### 배경
- **시간:** 오후 2:55
- **장소:** 골목 안 작은 카페 '문라이트'
- **분위기:** 빈티지, 조용함, 손님 2-3명
- **BGM:** 잔잔한 재즈

### 내레이션
```
카페를 찾았다.

진짜 숨어있는 곳이었다.
골목 안쪽, 간판도 작게.

문을 열었다.

띠링—

안은 어둑어둑하고 아늑했다.
손님은 몇 명 없었다.

그리고—

창가 구석.
검은 마스크에 후드티.

...있었다.
```

### Jun 발견
```
그가 고개를 들었다.

마스크 위로 눈이 보였다.

...웃고 있었다.

눈이 반달 모양으로 휘어졌다.

손을 살짝 들어 흔들었다.

심장이 미친 듯이 뛰었다.
```

"모르겠어요."

"이상하죠? 저도 알아요."

## SCENE 4: 대화

### 배경
- **시간:** 오후 3:20
- **상황:** 커피 마시며 대화
- **BGM:** 조용한 피아노

### 대화 - 일상
```
"요즘 뭐 하며 지내요?"

Jun이 물었다.

"그냥... 평범하게요."

"평범하게?"

"일하고, 밥 먹고, 자고."

"..."

그가 턱을 괴었다.

"부러워요."

"뭐가요?"

"평범한 거."

"..."

"저는 평범한 게 없어요."

그가 창밖을 봤다.

"아침에 일어나면 스케줄이 있고,
밥도 정해진 거 먹고,
잠도 스케줄 사이에 자고."

"..."

"평범하게 카페 오는 것도
이렇게 숨어서 와야 하고."

그가 나를 봤다.

"당신이랑 만나는 것도—"

### 선택지 3
```

```

**분기:**
- A → Jun 감동 (직접적 수락)
- B → Jun 솔직한 속마음 토로

## SCENE 6: 노을

- **분위기:** 노을, 주황빛, 사람 없음
- **BGM:** 감성 피아노 클라이맥스

```

Jun이 마스크를 벗었다.

"여기는... 괜찮겠다."

주변에 사람이 없었다.

그가 하늘을 봤다.

"예쁘다."

"네."

"..."

그가 나를 봤다.

"근데 더 예쁜 게 있어요."

"뭔데요?"

"..."

그가 웃었다.

"안 알려줄 거예요."

"뭐예요."

"비밀이에요."
```

### 분위기
```

...잘생겼다.

조명 없이, 카메라 없이.

"뭘 봐요?"

"..."

"왜 그렇게 봐요?"

"그냥요."

"예뻐서요."

하지만 내가 본 건 노을이 아니었다.
