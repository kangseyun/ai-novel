# EP5: 질투의 맛
## "The Taste of Jealousy"

### 에피소드 정보
- **분류:** 💎 유료 (50 하트)
- **예상 시간:** 15분
- **핵심 감정:** 질투, 불안, 달콤한 집착

### 핵심 줄거리
```
EP4 이후 2주.
Jun과의 연락은 계속되지만, 만남은 어렵다.

어느 날, 내 SNS에 회사 동료(남자)와 찍은 사진을 올린다.
점심 회식 사진. 별 의미 없는.

하지만 Jun이 그걸 봤다.

📱 [준]
그 사람 누구예요
옆에 있는 사람
왜 그렇게 웃어요
나한테는 그렇게 안 웃는데

질투하는 Jun.
처음 보는 감정 폭발.

"저 사람 때문에 화난 거 아니에요.
 당신 옆에 내가 아닌 사람이 있어서 화나는 거예요."
```

### 핵심 장면
1. **SNS 질투** - 문자 연타, 읽씹 시 전화
2. **감정 폭발** - "저만 이러는 거예요? 당신은 아닌 거예요?"
3. **화해** - "미안해요. 저도 이런 감정 처음이에요."
4. **달콤한 집착** - "다른 남자 사진 올리지 마요. 제발."

### 선택지 하이라이트
```
💭 Jun이 질투하고 있다.

→ [A] "질투해요?"
→ [B] "그냥 동료예요, 왜 그래요?"
→ [C] "...당신도 여자 팬들이랑 잘 웃던데요?" 💎

C 선택 시: Jun 격한 반응
"그건 팬서비스예요!
 진심 아니에요!
 당신한테만 진심이에요!"
```

### 엔딩 훅

- **분류:** 💎 유료 (60 하트)
- **예상 시간:** 18분
- **핵심 감정:** 설렘, 긴장, 달콤함
- **중요:** ⭐ 첫 키스 에피소드

### 핵심 줄거리
```
Jun의 숙소 방문.
처음으로 그의 "진짜 공간"에 들어간다.

깔끔하지만 텅 빈 느낌의 방.
```

## "The Choice"

- **분류:** 💎 유료 (80 하트)
- **예상 시간:** 20분
- **핵심 감정:** 갈등, 슬픔, 결단
- **중요:** ⭐ 엔딩 분기 결정

```

"제가 결정해야 하는 거잖아요.
 근데 당신 의견도 듣고 싶어요."

진지한 대화.
Jun은 세 가지 선택지를 말한다.

1. 헤어지고 아이돌로 남는다
2. 열애를 인정하고 그룹을 떠난다
3. 일단 부인하고 더 조심히 만난다

"당신은 뭘 원해요?"
```

3. **세 가지 선택** - 유저가 결정
4. **각오** - 선택에 따른 Jun의 반응

### 선택지 (엔딩 결정)
