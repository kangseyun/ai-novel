# EP1: 그 새벽, 편의점에서
## "Midnight Encounter"

---

## 에피소드 정보
- **분류:** 🆓 무료 에피소드

## SCENE 1: 잠 못 드는 밤

### 배경
- **시간:** 새벽 2:47 AM
- **장소:** 내 방
- **날씨:** 늦가을, 차가운 바람
- **BGM:** 없음 (정적)

### 내레이션
```
또 잠이 안 온다.

천장만 멍하니 바라보다가,
시계를 봤다. 새벽 2시 47분.

...나가자.
어디든.
```

### 선택지 1
```
💭 어디로 갈까?

→ [A] 한강 산책로
→ [B] 집 앞 편의점
→ [C] 그냥 누워있자
```

**분기:**
- A/B 선택 → SCENE 2로 진행 (같은 결과, 다른 묘사)
- C 선택 → 짧은 내레이션 후 결국 나가게 됨

### C 선택 시 추가 내레이션
```
이불을 끌어당겼다.

...

5분 뒤, 나는 신발을 신고 있었다.
가만히 있으면 미칠 것 같았으니까.
```

## SCENE 2: 새벽 편의점

### 배경
- **시간:** 새벽 3:12 AM
- **장소:** 한강변 24시 편의점 (CU/GS25 느낌)
- **분위기:** 형광등 불빛, 텅 빈 거리, 살짝 쌀쌀함
- **BGM:** lo-fi 비트 (희미하게)

### 내레이션
```
편의점 문을 열었다.

띵동—

알바생은 카운터에서 졸고 있었다.
나 말고는 손님이...

...있었다.

창가 테이블.
후드를 깊게 눌러쓴 남자가
라면을 먹고 있었다.
```

### 묘사
```
검은 후드티.
마스크.
고개를 푹 숙이고.

뭔가... 숨어있는 것 같은 자세.

이 시간에 라면이라.
나랑 비슷한 사람인가.
```

### 선택지 2
```
💭 그 남자가 신경 쓰인다.

→ [A] 슬쩍 쳐다본다
→ [B] 무시하고 음료 고르러 간다
→ [C] 같은 창가 자리에 앉는다
```

**분기:**
- A → SCENE 3A (눈 마주침 먼저)
- B → SCENE 3B (Jun이 먼저 말 걸어옴)
- C → SCENE 3C (바로 대화 시작)

...예쁘다.

남자인데. 예쁘다니.

하지만 그게 첫인상이었다.
반짝이는, 이상하게 슬퍼 보이는 눈.

```

## SCENE 4: 대화의 시작

### 배경
- **시간:** 새벽 3:20 AM
- **상황:** 둘 다 창가 테이블에 앉아 있음
- **BGM:** lo-fi 비트 살짝 커짐

### Jun 대사
```
잠시 후, 그가 먼저 입을 열었다.

"...여기 자주 와요?"

평범한 질문인데,
목소리가 이상하게 떨리는 것 같았다.
```

### 선택지 3
```
💭 대답할까?

→ [A] "아뇨, 처음이에요."
→ [B] "잠이 안 와서요. 당신은요?"
→ [C] "...모르는 사람한테 말 안 거는 편인데."
```

**분기:**
- A → 담담한 대화 진행
- B → Jun이 관심 보임 (호감도 +)
- C → Jun이 웃음 (희귀한 반응, 호감도 ++)

### 대화
```

그가 고개를 끄덕였다.

"저도... 처음은 아닌데,
자주 오는 건 아니에요."

어색한 침묵.

하지만 불쾌하진 않았다.
```

→ SCENE 5로 이동

"요즘 잠을 잘 못 자요."

그가 라면 국물을 저으며 말했다.

"그래서 가끔...

```

### 내 생각
```

```

## SCENE 5: 라면과 대화

### 배경
- **시간:** 새벽 3:35 AM
- **상황:** 편의점 라면을 같이 먹게 됨
- **BGM:** 잔잔한 피아노 멜로디

### 내레이션
```
어쩌다 보니
나도 라면을 사 왔다.

"같이 먹어도... 돼요?"

내가 물었더니
그가 고개를 끄덕였다.

마스크를 벗을까 말까
망설이는 것 같더니—

"저... 얼굴 좀 이상해도 신경 쓰지 마세요."

그가 천천히 마스크를 내렸다.
```

### Jun 외모 첫 공개
```
...

잘생겼다.

이상한 게 아니라
너무 잘생긴 거였다.

날카롭지만 부드러운 이목구비.
창백한 피부.
살짝 부은 눈.

아이돌 같다, 는 생각이
스쳐 지나갔다.

하지만 그냥 그런가 보다 했다.
요즘 잘생긴 사람 많으니까.
```

### 선택지 4
```
💭 뭐라고 할까?

→ [A] "...이상하진 않은데요?"
→ [B] "잘생겼네요." (솔직하게)
→ [C] 아무 말 없이 라면 먹기
```

**분기:**
- A → Jun 당황 (예상 외 반응)
- B → Jun 더 당황 + 귀 빨개짐 (호감도 +)
- C → Jun이 편안함을 느낌 (다른 방향 호감도 +)

## SCENE 6: 이름

### 배경
- **시간:** 새벽 4:05 AM
- **상황:** 라면 다 먹음, 대화 끝자락
- **BGM:** 희미해지는 lo-fi

### 대화
```
"근데..."

그가 빈 라면 용기를 바라보며 말했다.

"이름도 모르면서
같이 라면 먹었네요."

"그러게요."

잠시 침묵.

그가 먼저 말했다.
```

### Jun 자기소개
```
"저는..."

그가 잠시 망설였다.

"...준이에요. 그냥 준."

짧게 웃었다.

"근데 이름 물어봐도 돼요?"
```

### 선택지 5
```
💭 이름을 알려줄까?

→ [A] 이름을 말해준다 (유저 입력)
→ [B] "비밀이에요."
→ [C] "먼저 성 알려주면요."
```

**분기:**
- A → Jun이 이름 기억함 (중요 플래그)
- B → Jun이 아쉬워하며 재회 떡밥 ("그럼 다음엔...")
- C → Jun이 웃으며 "이 씨요." (본명 힌트, 복선)

## SCENE 7: 헤어짐

### 배경
- **시간:** 새벽 4:20 AM
- **상황:** 편의점 밖, 헤어지기 직전
- **BGM:** 피아노 + 새벽 바람 소리

### 내레이션
```

"저 가봐야 해요."

준이 시계를 보며 말했다.
얼굴에 아쉬움이 서렸다.

"이 시간까지 밖에 있으면 안 되는데..."
```

### 선택지 6
```
💭 헤어지기 전에

→ [A] "연락처 교환할까요?"
→ [B] "또 만날 수 있을까요?"
→ [C] "그럼 조심히 가요." (담담하게)

**분기:**
- A → Jun이 망설이다가 결국 번호 줌 (큰 분기)
- B → Jun이 떡밥 투척 ("...인연이면 만나겠죠")
- C → Jun이 아쉬워하며 떠남 (EP2에서 Jun이 적극적으로 찾아옴)

## SCENE 7 - A: 연락처 교환

### 대화
```
"연락처 교환할까요?"

그가 멈칫했다.

"..."

긴 침묵.

그리고—

"...진짜요?"

되물었다.
뭔가 믿기지 않는다는 얼굴.

"그냥... 이런 데서 만난 사람인데."

"그래서요?"

"..."

그가 주머니에서 폰을 꺼냈다.

손이 살짝 떨리는 것 같았다.

"아, 근데 이거..."

그가 화면을 가리며 말했다.

"비밀로 해줘요. 번호."

왜 그래야 하는지 몰랐지만,
고개를 끄덕였다.
```

### 훅
```
그의 전화번호를 저장했다.

'준'

단 한 글자.
그게 그에 대해 아는 전부였다.

집으로 돌아가는 길,
햇살이 떠오르고 있었다.

그리고 그때는 몰랐다.

그 '준'이
전국민이 아는 그 사람이라는 걸.
```

집으로 돌아가는 길,
마음이 이상하게 두근거렸다.

'인연이면 만나겠죠.'

그 말이 자꾸 맴돌았다.

그리고 며칠 후—
인연은 생각보다 빨리 찾아왔다.
```

## SCENE 7 - C: 담담한 이별

### 대화
```
"그럼 조심히 가요."

담담하게 말했다.

그런데 그가
살짝 당황한 것 같았다.

"아... 네."

뭔가 더 말하고 싶은 것 같았지만,
결국 입을 다물었다.

"그럼..."

그가 돌아섰다.

그런데 두 발짝 걷다가—
다시 돌아봤다.

"저기."

"네?"

"이름... 안 물어보는 거예요?"

"...아까 물어봤잖아요."

"내 말고. 당신 이름."

"..."

내가 대답하기 전에
그가 먼저 웃었다.

"다음에 알려줘요."

그러고는 진짜로 떠났다.
```

### 훅
```
다음에.

그 말이 약속처럼 남았다.

집으로 돌아가는 길,
그의 얼굴이 자꾸 떠올랐다.

잘생긴 얼굴.
외로워 보이는 눈.
살짝 부은 눈두덩이.

'새벽에 라면 먹던 남자.'

그게 내가 아는 전부였다.

그리고 며칠 후—
나는 그가 누구인지 알게 됐다.

TV 속에서.
```
